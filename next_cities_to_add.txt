Step-by-Step Guide to Add a New City
1. Add City to DATA_IMPORT_CONFIG in maps/config.py:
# Example: Adding a new city "Mumbai" to Maharashtra
DATA_IMPORT_CONFIG = {
    'states': {
        # ... existing states ...
        'maharashtra': {
            'name': 'Maharashtra',
            'code': 'MH',
            'cities': {
                'mumbai': {
                    'name': 'Mumbai',
                    'center_lat': 19.0760,
                    'center_lng': 72.8777,
                    'zoom_level': 11,
                    'data_format': 'geojson',  # or 'esri_json'
                    'layer_groups': {
                        'master-plan': {
                            'name': 'Mumbai Master Plan',
                            'description': 'Mumbai Development Plan',
                            'path': 'maharashtra/mumbai/master_plan',
                            'display_order': 1,
                            'files': {
                                'Residential_Zone.json': {
                                    'name': 'Residential Zone',
                                    'color': '#FFC400',
                                    'category': 'RESIDENTIAL'
                                },
                                'Commercial_Zone.json': {
                                    'name': 'Commercial Zone', 
                                    'color': '#004DA8',
                                    'category': 'COMMERCIAL'
                                },
                                'Industrial_Zone.json': {
                                    'name': 'Industrial Zone',
                                    'color': '#C500FF',
                                    'category': 'INDUSTRIAL'
                                }
                                # ... add more files as needed
                            }
                        }
                    }
                }
            }
        }
    }
}

2. Add Color Mapping to _get_feature_color Method
You need to add the new city's color mapping to both generate_local_png_tiles.py and generate_direct_s3_tiles.py:
Add your new city's color mapping like this:
# Add this to the _get_feature_color method in both files
elif city_slug == 'mumbai':
    # Use pattern style function for Mumbai
    source_layer = properties.get('source_layer_name', '').strip()
    if source_layer:
        try:
            # Try with the source layer name first
            pattern_style = get_pattern_style('mumbai', source_layer)
            if not pattern_style:
                # Try with filename pattern
                filename_pattern = source_layer.replace(' ', '_').replace('___', '_').replace('__', '_') + '.geojson'
                pattern_style = get_pattern_style('mumbai', filename_pattern)
            if pattern_style:
                if 'hatch' in pattern_style:
                    return {
                        'fill_color': pattern_style.get('solid', '#FFFFFF'),
                        'stroke_color': '#2C3E50',
                        'pattern': 'HATCHED',
                        'hatch_color': pattern_style['hatch']
                    }
                elif 'solid' in pattern_style:
                    return {
                        'fill_color': pattern_style['solid'],
                        'stroke_color': '#2C3E50',
                        'pattern': 'SOLID'
                    }
        except:
            pass
        
        # Fallback to direct mapping
        mumbai_file_colors = {
            'Residential Zone': '#FFC400',
            'Commercial Zone': '#004DA8',
            'Industrial Zone': '#C500FF',
            'Mixed Use Zone': '#FFAA00',
            'Transport Zone': '#828282',
            'Green Zone': '#55FF00',
            'Water Body': '#73FFDF'
        }
        
        # Direct match first
        if source_layer in mumbai_file_colors:
            return {
                'fill_color': mumbai_file_colors[source_layer],
                'stroke_color': '#2C3E50',
                'pattern': 'SOLID'
            }

Summary of Files to Modify:
maps/config.py - Add city to DATA_IMPORT_CONFIG
maps/management/commands/generate_local_png_tiles.py - Add color mapping
maps/management/commands/generate_direct_s3_tiles.py - Add color mapping (same as local)
Excel file - Add new state/city to hierarchy
The system is designed to be easily extensible - just follow the pattern of existing cities and add your new city's configuration!
